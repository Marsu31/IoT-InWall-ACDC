# IoT-InWall-ACDC


Ce montage est dérivé du projet [HLK-PM01 breakout board](https://www.openhardware.io/view/504/HLK-PM01-breakout-board). Ce dernier avait l'inconvénient de ne pas rentrer dans les boitiers muraux en France. J'ai donc décidé d'adapter l'implantation des composants.
